## Website Goal

Just a minimilast but modern static informational landing page, SEO optimized so clients can find the company on search engines. No need for backends or CMS features to create custom content, if a change is needed in the future it can be manually edited in the code, constant changes are not expected. The website is supposed to be a simple continuous landing page.

## Languages

The website should use be able to detect the users language and auto select the best match and use an extensible translations system. It should have the following languages available:

- French from France (default fallback)
- English
- Italian
- German
- Spanish
- Dutch
- Portuguese, European from Portugal

Use separate files for each language translation key - values.

## Brand Identity
